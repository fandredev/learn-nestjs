## Clone this repo

###### Notes: Check the .env.example before clone this repo

```
git clone git@github.com:fandredev/concepts_nest.git
```

## Go to directory

```
 cd concepts_nest
```

## Install dependencies

```
 pnpm i
```

## Run server

```
 pnpm run start:dev
```

## Run linters

```
pnpm run format && pnpm run lint
```
